Pour qui est fait le Mucar CS90 ?

Contrairement au VO7 qui vise le diagnostic "tous systèmes", le CS90 est un outil spécialisé pour la maintenance courante. Il ne diagnostique "que" les 4 systèmes vitaux (Moteur, ABS, Airbag, Boîte de vitesses), ce qui est suffisant pour 80% des pannes. Mais sa force réside ailleurs : il offre une couverture quasi-totale des fonctions de réinitialisation.

FAQ

Fonctionne-t-il si je n'ai pas de Wi-Fi au garage ?

Oui pour le diagnostic ! Le Wi-Fi n'est nécessaire que pour les mises à jour ou pour envoyer le rapport par email. Une fois le logiciel la marque téléchargé, vous pouvez travailler hors ligne.

Peut-il coder une nouvelle batterie ?

Oui, la fonction BMS (Battery Management System) permet d'enregistrer une nouvelle batterie lors de son remplacement, indispensable sur les voitures modernes (Start&Stop).

Je ne vois pas les 28 fonctions sur l'écran d'accueil ?

C'est normal. Les fonctions de reset sont souvent accessibles directement via le menu "Maintenance" ou s'affichent intelligemment une fois le véhicule sélectionné.
