Comment effacer un code défaut (DTC) ?

Comprendre le code c'est bien, pouvoir réinitialiser le voyant c'est mieux.

Un dongle ELM327 basique lira le P0420 mais échouera à l'effacer sur les véhicules récents dotés d'une passerelle SGW. Pour lire et effacer tout l'électronique de bord sans ruiner l'ECU, nous recommandons la gamme Mucar ou Thinkcar.
